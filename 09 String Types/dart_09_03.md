```Dart
String s1 = "My name is ";
String s2 = "Dr.Sungwon";
String s3 = s1 + s2;
```
\+ 연산을 통하여 문자열들을 하나로 합칠 수 있다.

```Dart
var s4 = 'My '
         'name '
         'is '
         'Dr.Sungwon';
```
여러 문자열을 나열해도 하나로 합쳐짐을 알 수 있다.
```Dart
dynamic s5 = '''
My name is Dr.Sungwon.
Dart is lovely.
''';
```
\'를 3번 쓴 '''를 이용하여 여러 줄의 문자열을 저장할 수 있다.
